# Extension Icons

Replace these placeholder files with actual PNG icons before publishing.

## Required Sizes

- `icon-16.png` - 16x16 pixels (toolbar, favicon)
- `icon-32.png` - 32x32 pixels (Windows taskbar)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store, installation)

## Design Tips

- Use a simple, recognizable design
- Works well at small sizes
- Use transparent background
- Use consistent colors with your brand

## Tools to Create Icons

- Figma (free)
- Canva
- GIMP
- Adobe Illustrator
- Online favicon generators

## Quick Placeholder

You can use this SVG and convert to PNG at different sizes:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 128 128">
  <rect width="128" height="128" rx="16" fill="#2563eb"/>
  <text x="64" y="80" text-anchor="middle" fill="white" font-size="64" font-weight="bold">V</text>
</svg>
```
